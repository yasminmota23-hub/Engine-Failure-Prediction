## 📖 What is Engine-Failure-Prediction?

This application helps you detect engine problems before they become serious. It uses sensors connected to an ESP32 device to track engine data like temperature, vibration, pressure, and RPM. The app then uses machine learning models to analyze this data in real time and predict potential engine failures. This helps you avoid breakdowns and save on maintenance.

You do not need any programming knowledge to use this software. The app runs on Windows and works with simple setup steps described below.

## 🔌 Connecting to the ESP32 Sensors

You will need an ESP32 device that gathers engine data via sensors. 

1. Connect the ESP32 device to your computer using a USB cable.  
2. Turn on the ESP32 device and ensure it is powered properly.  
3. The application will recognize the ESP32 automatically when it runs. If not, check the USB connection or try a different port.  

The software communicates with these sensors to collect live data for analysis.

## 🧰 Troubleshooting

- **Program won’t start:** Check if your Windows version meets the requirements and all updates are installed.  
- **ESP32 device not detected:** Ensure the USB cable is working and connects firmly. Restart the device and software.  
- **No data on screen:** Make sure sensors are properly connected to the ESP32 device.  
- **Installation failed:** Try running the installer as Administrator by right-clicking and selecting “Run as administrator.”  

If you continue having issues, check the FAQ section on the GitHub page or contact support through the issues tab.
